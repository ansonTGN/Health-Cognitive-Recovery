# Health-Cognitive-Recovery
Sistema RAG semántico especializado en salud mental comunitaria. Fusiona Rust, Neo4j e IA para transformar datos clínicos y sociales en un grafo de conocimiento vivo. Facilita el análisis del Modelo de Recuperación, conecta intervenciones con resultados y garantiza la interoperabilidad ética de datos mediante ontologías estándares.
